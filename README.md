# Asel Proje Website

Modern corporate website project built with React, TypeScript and Vite.

## Overview

Asel Proje Website is a responsive frontend project designed for a construction, excavation and marine services brand.  
The project focuses on clean UI design, service presentation, visual branding and a modern web experience.

## Features

- Responsive React frontend
- TypeScript-based project structure
- Vite development environment
- Modern landing page layout
- Project/service showcase sections
- Custom visual assets
- Clean and maintainable component structure

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- HTML

## Project Structure

```text
src/
  App.tsx
  App.css
  main.tsx
  index.css

public/
  assets/
    project and brand images
````

## Running Locally

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Project Status

This project was created as a portfolio and business website demo.
